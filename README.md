# Dynamic Color Changer 🎨

A simple interactive web project that allows users to change the background color and text size dynamically using JavaScript.

This project was built as part of revising core front-end development concepts.

## 🚀 Tech Stack

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## ✨ Features

* Change background color with a button click
* Increase text size dynamically
* Reset button to restore default styles
* Simple and interactive UI

## 📚 What I Practiced

* DOM selection using `querySelector`
* Event handling with `addEventListener`
* Manipulating styles using JavaScript
* Writing clean and simple interactive logic

## 📌 Future Improvements

* Add multiple color options (random color generator)
* Smooth transitions and animations
* Improve UI with better styling
* Add more interactive controls

## 📁 Project Status

Completed as a basic revision project to strengthen JavaScript fundamentals and interactivity.
